# Пул эталонных изображений

Эта папка предназначена для набора эталонных изображений (шаг 3 алгоритма
выполнения проекта — «Формирование пула эталонных изображений для оценки
качества работы системы»).

Положите сюда несколько изображений в поддерживаемых форматах (JPG, PNG,
HEIC, BMP), желательно разного характера:

- недоэкспонированное (тёмное) фото;
- переэкспонированное (пересвеченное) фото;
- малоконтрастное/блёклое фото (туман, дымка);
- малонасыщенное фото (пасмурная погода);
- уже хорошо снятое фото (проверка, что автокоррекция не портит его).

Папка уже содержит стартовый набор из 6 изображений (`01`–`06`),
сгенерированный из реальных фотографий тестового набора Kodak24 —
по одной паре «хорошее фото → известная синтетическая деградация» на
каждую категорию выше (см. `tools/train-tone-model/generate-samples.js`).
А также фото размером 15Мб из открытых источников
